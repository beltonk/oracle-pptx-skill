**Dark theme**: `resources/icons/dark-theme/`
**Light theme**: `resources/icons/light-theme/`

**File naming:** `IconName_Bark_RGB.svg` (dark) or `IconName_Air_RGB.svg` (light)

## Finding Icons

**Search by keyword:**
```bash
grep -i "analytics" resources/icons/ICON-LIBRARY.md
grep -i "cloud" resources/icons/ICON-LIBRARY.md
```

**Common icon mappings:**
- **Finance/Money**: Analytics, Billing, Budget icons (Business category)
- **Cloud/Infrastructure**: OCI, Cloud, Container icons (Technology/OCI categories)
- **Data/Analytics**: AI-Database, Big-Data, Analytics icons (Technology/Business)
- **Security**: Security, Identity-Management icons (Technology category)
- **People/Teams**: Personas category (Developer, Manager, Customer, etc.)
- **Processes**: Calendar, Forecast, Integration, Workflow icons (Business)
